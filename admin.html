SETUP

Googleスプレッドシートで使う流れ

1

ExcelをGoogleドライブへアップロード

管理用スプレッドシートをGoogleドライブに入れて、Googleスプレッドシートとして開きます。

2

各シートをCSV公開

「在庫車両」と「ご提案例」をそれぞれCSV形式でウェブ公開します。

3

CSV URLを設定

script.js のURL設定を、GoogleスプレッドシートのCSV公開URLに差し替えます。

設定する場所 
const stockDataUrl = "在庫車両シートのCSV公開URL";
const exampleDataUrl = "ご提案例シートのCSV公開URL";
